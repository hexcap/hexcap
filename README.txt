A simple curses based hex editor for pcap files.
hexcap requires the python libraries libdnet and pcapy

To install hexcap:
1) fork the repo from github
2) install python extension for libdnet 
  On OpenBSD this is py-libdnet-1.12p9:python interface to libdnet
3) install python extension for pcap(pcapy), not pypcap
  On OpenBSD this is py-pcapy-0.10.5p6:libpcap module for Python
4) ???
5) Profit!
